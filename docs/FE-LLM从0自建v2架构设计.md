# FE-LLM 从 0 自建 v2 架构设计

更新日期：2026-06-13

> 这份文档是 FE-LLM 的主线蓝图：一个**完全从 0 构建、不依赖任何预训练底座**的认知架构。
> 命名一律用标准 ML / 认知科学术语，东方哲学（道/易）只作为设计直觉的注释，不进代码符号。
> 上游依据：`FE-LLM规范架构方案.md`（模块基线）、`FE-LLM道易主动推理方案草案.md`（愿景）、
> `想法对话.md`（分层预测编码）、`经验.md`（全部阴性/阳性教训）。

## 0. 这份设计要回答的问题

用户的诉求：**脱离预训练底座，从 0 训练我们自己的架构，哪怕简单，但必须是自己的。**

v2 的定位不是"再借一个 Qwen"，而是：

> 把"能量解释 / 预测误差 / 自由能平复 / 主动推理 / 可溯源 / 自我成长"这六要素，
> 落成一个小而自洽、从 0 训练、能跑能成长的完整体。

## 1. 必须先认清的边界（吸取教训，避免再"搞半天"）

从 `经验.md` 两次阴性 + 一串底座旁支实验，提炼一条**硬纪律**：

> **v2 不和通用 LLM 拼"世界知识 / 翻译 / 开放问答"。**
> 这些能力需要海量参数承载，8.9M 从 0 模型物理上装不下（翻译 word-F1 0.07/0.08 已证伪）。

同样的从 0 架构，在 FE-LLM **核心机制**上已有阳性证据：

| 能力 | 证据 | 结论 |
|---|---|---|
| 主动推理动作选择 | 实验 C：FE-agent 100% vs baseline 22% | 从 0 可行 |
| 动作分类 | policy balanced accuracy 0.929 | 从 0 可行 |
| 多轮 surprise 平复 | 实验 B：澄清后降幅 70%~88% | 从 0 可行 |
| 意图驱动短生成 | dialogues decode_loss 1.82 | 从 0 可用（受限） |
| 翻译 / 全句重建 | word-F1 0.07~0.08 | 从 0 **不可行**（容量） |

**v2 战场 = 前四行（机制），不是最后一行（知识）。**

## 3. v1 现状 → v2 目标

### v1 现状（已是自建、从 0、无底座）

- `fe_llm/energy_lm/`：字符级 tokenizer + `IntentEncoder`(PER 块) + `EnergyDecoder`(能量递减解码)，8.9M，从 0 训练；
- `fe_llm/active_inference/`：observation → perception → prediction → surprise → belief_update → policy → free_energy → action → trace → memory 全闭环。

### v1 短板（v2 要补）

1. **意图是单个全局向量**：`IntentEncoder` 取 [CLS] 得 `z ∈ R^d`，无"象/卦/爻"层级（道易草案第 6 节要求层级）。
2. **预测编码只在序列内**：`PERBlock` 的预测-误差-弛豫在 token 序列内部，不是**跨抽象层级**的自上而下预测 + 自下而上误差（规范架构方案 2.1 要求 `z_1..z_L`）。
3. **surprise 不分层**：当前 5 通道 surprise 是控制层标量，没和生成层分层 latent 误差打通。

### v2 一句话目标

> 把"单层全局意图 + 序列内 PER"升级为**真正的分层预测编码世界模型**（`z_1..z_L`，
> 自上而下预测、自下而上误差、自由能弛豫到收敛），并让分层误差直接喂给 surprise 与 policy。

## 4. v2 核心引擎：分层预测编码（HPC）

### 4.1 数学形式

维护 L 层 latent `z_1 ... z_L`（z_1 底层=词形/局部，z_L 顶层=抽象意图/卦）：

```text
自上而下预测：  z_hat_l = g_l(z_{l+1})        # 高层预测低层
预测误差：      eps_l   = z_l - z_hat_l
精度加权自由能：F = sum_l  0.5 * eps_l^T * Pi_l * eps_l   # Pi_l 为该层精度
感知即弛豫：    z_l <- z_l - alpha * dF/dz_l    # 迭代到 F 收敛
```

这是 `想法对话.md` 说的"上下层互相妥协，直到自由能归零"的工程化，也是
v1 论文留的升级口子（belief update 升级为多层 PC relaxation）。

### 4.2 与现有 PERBlock 的关系

`PERBlock` 已有雏形：`mu = W_pred(h)`（预测）、`err = h - expect`（误差）、`z = z - eta*err`（弛豫）。
v2 不推倒它，而是**从"序列内单层"提升为"跨抽象层级"**：v1 一层内 token 互相预测；
v2 层与层之间预测（`z_{l+1}` 预测 `z_l`），保留序列内 PER 作为每层内部算子。

### 4.3 对应道易（仅注释）

| 道易概念 | 标准术语 | 张量 |
|---|---|---|
| 卦（全局态势） | top latent | `z_L` |
| 爻（局部要素） | mid latents | `z_2..z_{L-1}` |
| 爻位精度 | precision | `Pi_l` |
| 观象 | bottom-up encoding | `z_1 = enc(x)` |
| 穷则变 | 结构成长触发 | 见第 7 节 |

## 5. v2 模块（落到 `fe_llm/`，对齐规范架构方案）

```text
fe_llm/
  perception/   # 已有 → 补:分词+嵌入+证据路由
  world_model/  # 【新】分层 latent z_1..z_L + 先验
  predictive/   # 【新】自上而下预测 g_l + 自下而上误差 eps_l + 弛豫
  surprise/     # 已有 → 接分层误差
  situation/    # 【新】把 z_L/误差压成结构化情境(象/卦/爻接口)
  policy/       # 已有 保留
  free_energy/  # 已有 保留
  controller/   # 已有 保留, 串新世界模型
  decoder/      # 已有 EnergyDecoder → 条件于 z_1..z_L
  memory/       # 已有 保留
  trace/        # 已有 → 记录分层误差
```

迁移策略：**逐模块替换，不推倒重来，每步保持可运行可评估。**

## 6. 训练路线与判定（每步可判定，不靠感觉）

- **阶段一**：`world_model + predictive + decoder` 从 0 学最小重建/生成；数据用 `dialogues.jsonl`（不碰翻译）；判定：分层版 decode_loss 不劣于 v1（≈1.82）。
- **阶段二**：分层误差喂 surprise/situation/policy；判定：动作选择 bal_acc≥0.929、实验 B 5/5、实验 C 不回退。
- **阶段三**：成长（短期 belief / 长期 memory / 离线再训练）；判定：记忆读回不回退 + 审计日志。

## 7. 自我成长（三层，"穷则变"）

1. 短期：对话内 belief 更新（已有）；
2. 中期：长期记忆候选队列，可审计（已有雏形）；
3. 长期：仅当分层自由能长期居高（原结构解释力到极限）才触发离线结构成长。

## 8. 第一个里程碑 V2-M1（最小可判定切片）

**不要一次铺满 11 个模块。** 第一步只做一件能证明分层有价值的事：

> 把 `IntentEncoder` 单层全局向量升级为 **2 层 latent**：`z_global`(摘要) + `z_local`(序列)，
> `z_global` 自上而下预测 `z_local` 统计量，`z_local` 自下而上修正 `z_global`，弛豫若干步到自由能收敛。

### V2-M1 判定（三条都要过）

1. **不破坏生成**：2 层版 decode_loss ≤ v1 单向量版（同数据同预算）；
2. **分层承载不同粒度**：线性探针下 `z_global` 预测意图/动作类别准确率显著高于 `z_local`，而 `z_local` 预测局部 next-token 准确率显著高于 `z_global`；
3. **分层 surprise 可定位**：语义异常样本 vs 词形异常样本，高层误差对语义更敏感、低层误差对词形更敏感（可区分）。

### V2-M1 停损

- decode_loss 明显劣化 → 弛豫/预测有 bug，先修；
- 探针显示两层无分工 → 加层间正交 / 信息瓶颈约束；
- 仍无分工 → 回第 4 节重审 HPC 形式，而不是硬堆层数。

## 9. 风险

- **分层塌缩**（类槽位塌缩）→ 层间正交 + 精度差异化；
- **弛豫不稳定** → 限制弛豫步数 + 残差范数裁剪（沿用 backbone 的 delta 范数约束教训）；
- **过度设计** → 严格按 V2-M1 → 阶段一/二/三推进，每步可评估；
- **又被知识任务带偏** → 回看第 1 节硬纪律。

## 10. 不变量（任何实现都不能违背）

| 要素 | v2 形态 |
|---|---|
| 能量解释 | 分层自由能 F = sum 精度加权误差，显式可读 |
| 预测误差 | 每层 eps_l 显式，分层定位惊奇来源 |
| 自由能平复 | 感知=弛豫到 F 收敛；行动=最小化期望自由能 |
| 主动推理 | 显式行动空间 + EFE，controller 闭环不变 |
| 可溯源 | trace 记录分层误差/精度/动作，不靠 attention 权重 |
| 自我成长 | 三层边界；穷则变才触发结构成长 |
