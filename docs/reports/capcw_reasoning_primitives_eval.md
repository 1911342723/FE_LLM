# CAPCW 检索之外的推理基元：in-context 比较 / 计数（flat vs CAPCW）

- 判定：**诚实负结果(无新增推理基元): CAPCW 在本规模未展现**检索之外**的新推理基元——compare 优势**容量依赖**（低 K 单向量够用、打平；仅高 K=7 容量受限时 CAPCW +0.130，均值 +0.0670<0.10 被低 K 平局拉低）——这是与**检索同源的容量效应**，非新增推理超能力；count **无 headroom**（flat 自身已达 1.000≈满——单向量用'值编码求和'即可计数，keys 无关，根本不测内容寻址）。结论：compare 本质是'取回两值再比'(优势=容量效应,与绑定/检索同源)，count 单向量可解(无 headroom)；CAPCW 的价值仍锚定在**容量受限的内容寻址取回/绑定**，未额外解锁比较/计数这类运算推理。**
- 设置：n_keys=12, n_vals=10, d=32, seeds=2；唯一变量=world 结构。
